# 🦾 Sentinel-6: Enterprise Digital Twin
> Una solución de mantenimiento predictivo que une el 3D de alta fidelidad con la analítica de datos industrial.

## 🎯 Resumen de Solución
Sentinel-6 no es solo un modelo 3D; es un sistema de monitoreo. Captura telemetría de sensores, procesa el estado de salud mediante IA y visualiza el comportamiento en un entorno USD (Universal Scene Description).

### 🛠️ Pilares del Proyecto
1. **Pipeline de Datos:** Motor en Python que emula sensores industriales.
2. **Technical Art:** Activo 3D optimizado con Rigging cinemático para sincronización con datos.
3. **Conectividad:** Integración con ecosistemas CRM (Salesforce) para flujo de trabajo reactivo.
